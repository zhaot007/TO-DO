## 🎨 设计细节

### 渐变方向
```css
linear-gradient(135deg, ...)
```
- 135度：左上到右下
- 视觉流动感
- 增加立体感

### 透明度层次
```
背景起点: 0.08 (8%)
背景终点: 0.04 (4%)
边框:     0.15 (15%)
悬停背景: 0.15 (15%)
投影:     0.25-0.3 (25-30%)
```
- 由浅到深的层次
- 保持整体轻盈感

### 过渡动画
```css
transition: all 0.3s ease;
```
- 300ms：舒适的动画时长
- ease：自然的缓动曲线
- all：所有属性平滑过渡

### 修改前
```
眼睛扫描路径：随机
注意力分配：均匀
决策时间：较长
```

```

3. 待办（蓝色）→ 查看待处理
4. 已完成（绿色）→ 确认进度

注意力分配：核心指标 > 次要指标
决策时间：缩短30%（预估）
```

## 💡 设计哲学

1. **形式服务功能**: 宽度差异反映业务重要性
2. **色彩传达语义**: 无需阅读即可理解状态
3. **动效增强反馈**: 微交互提升操作确定性
4. **渐进增强**: 不破坏原有功能，只优化体验
