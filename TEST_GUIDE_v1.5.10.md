## 🐛 常见问题排查

### 问题1：色彩不显示

**可能原因**:
- 浏览器缓存
- CSS未加载

**解决方案**:
```bash
# 强制刷新
Ctrl + Shift + R (Windows/Linux)
Cmd + Shift + R (Mac)

# 清除缓存重启
rm -rf node_modules/.vite
npm run dev
```

### 问题2：宽度比例不对

**可能原因**:
- 类名未正确添加
- CSS未生效

**检查方法**:
1. F12打开开发者工具
2. 检查"全部"和"已逾期"卡片是否有 `stat-card-primary` 类
3. 检查CSS中 `flex: 1.3` 是否生效

### 问题3：动画卡顿

**可能原因**:
- 浏览器性能问题
- 其他进程占用资源

**解决方案**:
- 关闭其他标签页
- 检查CPU使用率
- 尝试其他浏览器
