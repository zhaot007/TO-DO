## 📊 核心改动速览

### 输入框
```
修改前: 外阴影（浮起）
修改后: 内阴影（凹陷）+ 聚焦外发光
```

### 属性标签
```
修改前: 白色 + 边框 + 36px高
修改后: 浅灰胶囊 + 无边框 + 32px高
```

### 高度对齐
```
输入框:   36px
提交按钮: 36px（对齐）
属性标签: 32px（略矮）
```

---

## 🎨 关键CSS

### 内阴影
```css
box-shadow: 
  inset 0 2px 4px rgba(0, 0, 0, 0.06),
  0 1px 2px rgba(0, 0, 0, 0.04);
```

### 外发光
```css
box-shadow: 
  inset 0 1px 2px rgba(0, 0, 0, 0.04),
  0 0 0 3px rgba(102, 126, 234, 0.1);
```

### 胶囊样式
```css
background: rgba(0, 0, 0, 0.04);
border: none;
border-radius: 18px;
height: 32px;
```

## 🐛 问题排查

### 看不到内阴影
```bash
Ctrl + Shift + R  # 强制刷新
```
降低显示器亮度可以更清楚地看到

### 属性标签还是白色
检查 `.attr-group` 的 `background`  
应该是 `rgba(0, 0, 0, 0.04)`
