## 📊 核心改动速览

### 宽度比例
```
全部(1.3x) | 已完成(1x) | 待办(1x) | 已逾期(1.3x)
```

### 色彩方案
```
全部:   白色（中性）
已完成: 淡绿色（成功）
待办:   淡蓝色（进行中）
已逾期: 淡红色（警告）
```

### 微交互
```
默认 → 悬停(-2px) → 激活(-1px)
投影: 1px → 4px → 3px
```
