## ✅ 测试清单

### 1. 输入框内阴影测试（30秒）

**目标**: 验证凹陷质感

**步骤**:
1. 打开应用
2. 点击统计区域的"添加"按钮（↓图标）
3. 观察输入框

**预期结果**:
- ✅ 输入框有轻微的"凹陷"感（内阴影）
- ✅ 看起来像"嵌入"页面中
- ✅ 引导用户点击输入

**视觉检查**:
```
修改前: 输入框浮在页面上（外阴影）
修改后: 输入框嵌入页面中（内阴影）
```

**如何判断**:
- 仔细观察输入框顶部边缘
- 应该有轻微的暗色阴影向内
- 不是向外的投影

## 🐛 常见问题排查

### 问题1：看不到内阴影

**可能原因**:
- 浏览器缓存
- 显示器亮度过高

**解决方案**:
```bash
# 强制刷新
Ctrl + Shift + R (Windows/Linux)
Cmd + Shift + R (Mac)

# 调整显示器亮度
降低亮度可以更清楚地看到内阴影
```

### 问题2：属性标签还是白色

**可能原因**:
- CSS未加载
- 类名未更新

**检查方法**:
1. F12打开开发者工具
2. 检查 `.attr-group` 的 `background` 属性
3. 应该是 `rgba(0, 0, 0, 0.04)`

### 问题3：高度不对齐

**可能原因**:
- 浏览器缩放
- CSS未生效

**检查方法**:
1. 确保浏览器缩放为100%
2. F12检查元素高度
3. 输入框和按钮应该是36px
4. 属性标签应该是32px
