# v1.5.14 快速参考

## 🚀 一键启动

```bash
npm run dev
```

浏览器访问: `http://localhost:5173`

---

## ✨ 核心功能：智能任务描述

### 使用流程

1. **输入任务名称**
   ```
   ➕ 新建任务：完成项目报告
   ```

2. **描述输入框自动出现**（动画）
   ```
   📝 任务描述（可选）...
   ```

3. **可选输入描述**
   ```
   包含数据分析和可视化图表
   需要在周五前提交
   ```

4. **创建任务**
   - 按回车键
   - 或点击✓按钮

---

## ✅ 快速验证（30秒）

1. 点击添加按钮（↓）展开表单
2. 输入任务名称（如"买牛奶"）
3. 观察描述输入框是否自动出现
4. 尝试输入描述（可选）
5. 创建任务，查看任务列表

## 🎨 关键特性

### 智能显示
```javascript
v-if="newTaskText.trim()"
```
- 输入名称 → 自动出现
- 清空名称 → 自动隐藏

### 风格一致
- 内阴影凹陷感
- 聚焦紫色外发光
- 与名称输入框一致

```css

resize: vertical;
```
