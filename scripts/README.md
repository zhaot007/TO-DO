# 实用脚本 | Utility Scripts

本目录包含用于开发和测试的实用脚本。

## 📋 脚本列表

### 1. 清理本地数据.js
**用途**: 清空浏览器中的所有任务数据（包括回收站）

**使用场景**:
- 测试导入功能前清空旧数据
- 重置应用到初始状态
- 清理测试数据

**使用方法**:
1. 打开应用页面 http://localhost:5173/#/todo
2. 按 F12 打开浏览器开发者工具
3. 切换到 Console（控制台）标签
4. 复制 `清理本地数据.js` 的全部内容
5. 粘贴到控制台并按回车执行
6. 页面会自动刷新，所有任务数据被清空

**代码示例**:
```javascript
localStorage.setItem('CapacitorStorage.tasks', '[]');
localStorage.setItem('CapacitorStorage.deletedTasks', '[]');
location.reload();
```

### 2. 统计任务数据.js
**用途**: 在浏览器控制台中查看任务的详细统计信息

**使用场景**:
- 验证导入功能是否正常
- 检查任务数据分布
- 调试数据问题

**使用方法**:
1. 打开应用页面 http://localhost:5173/#/todo
2. 按 F12 打开浏览器开发者工具
3. 切换到 Console（控制台）标签
4. 复制 `统计任务数据.js` 的全部内容
5. 粘贴到控制台并按回车执行
6. 查看输出的统计信息

**输出示例**:

## ⚠️ 注意事项

1. **数据安全**: 清理脚本会永久删除所有本地数据，请谨慎使用
2. **浏览器限制**: 首次粘贴代码时，浏览器会要求输入"允许粘贴"以确认
3. **开发环境**: 这些脚本仅用于开发和测试，不要在生产环境使用
4. **存储位置**: 脚本操作的是 `CapacitorStorage.*` 键，而非普通的 `localStorage`
